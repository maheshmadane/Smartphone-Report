# Smartphone-Report
## Smartphone Report
### 1. Short Description
📌 Built an interactive Smartphone Analysis Dashboard using Power BI to analyze key features like brand, RAM, storage, chipset, and launch trends.

### 2. Tech Stack
⚙️ Power BI
⚙️ SQL
⚙️ Data Modeling
⚙️ DAX
⚙️ Power Query

### 3. Data Source
📂 Dataset collected from Kaggle (Smartphone Dataset)

### 4. Features / Highlights
📊 Brand-wise smartphone distribution analysis
📈 RAM and storage trend analysis
⚙️ Chipset (CPU) comparison across models
🎯 Interactive dashboard with slicers (brand, model, chipset, year)
📌 KPI visuals for quick insights
🔄 Data cleaning and transformation using Power Query

### 5. Example Insights
🔍 Identified top smartphone brands based on model count
📉 Analyzed trends in RAM and storage over the years
📊 Compared chipset performance across different brands
📈 Observed market trends and feature evolution
